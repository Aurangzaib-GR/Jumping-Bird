Welcome to my portfolio! **Jumping Bird** is a classic Flappy Bird-style 2D arcade game built in Unity. I developed this project to refine my skills in 2D gameplay mechanics, infinite scrolling, and WebGL optimization.

## 🕹️ How to Play
* **Mouse Click:** Click the left mouse button to make the bird jump and navigate through the gaps in the pillars.
* **Avoid Obstacles:** Do not crash into the green pillars or hit the ground!
* **Survive:** Keep flying through the gaps to test your reflexes and survive as long as possible.

## 🛠️ Development Highlights
* **Physics & Collisions:** Fine-tuned `Rigidbody2D` for responsive jumps and precise hitboxes.
* **Infinite Scrolling:** Implemented seamless background recycling and dynamic pillar spawning.
* **WebGL Optimization:** Configured forced portrait resolution (540x960) and compression fallbacks for fast browser loading.
* **Responsive UI:** Smooth screen flow and UI anchoring for the "Game Over" and "Retry" states.
